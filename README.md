# VolunteersMIS

> ### 基本介绍

基于JSP和Servlet开发，数据库使用Mysql，核心业务是社团志愿活动发布、报名和审核。用户有两种角色：普通志愿者和管理员

> ### 功能介绍

* 普通志愿者
  1. 志愿者用户注册页面
  2. 修改个人资料
  3. 查看所有志愿活动
  4. 查看报名表审核状态

* 管理员
  1. 发布志愿活动
  2. 审核注册志愿者申请名单
  3. 审核志愿活动报名表

> ### 项目特色

1. 使用监听器
2. 使用过滤器
3. 分页技术运用
